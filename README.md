# Smart-Camera

Real Time Camera Object Detection with Machine Learning(Core ML) using, Resnet50 in Swift. Resnet50 is an Image Classification Residual Neural Network that will classify and Detects the dominant objects present in an image from a set of 1000 categories such as trees, animals, food, vehicles, people and more in a camera frame or image.
